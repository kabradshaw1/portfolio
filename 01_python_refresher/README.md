# Python Refresher

Core Python exercises demonstrating proficiency with data structures, OOP, async programming, type hints, and data processing.

Each script is standalone — run it directly to see output and examples.

## Scripts

| File | Topic |
|------|-------|
| `data_structures.py` | Lists, dicts, sets, comprehensions, generators |
| `oop_patterns.py` | Classes, inheritance, ABCs, dataclasses |
| `async_basics.py` | asyncio, async/await, concurrent tasks |
| `type_hints.py` | Type annotations, generics, Protocol |
| `data_processing.py` | pandas/numpy data manipulation |

## Setup

```bash
pip install -r requirements.txt
python data_structures.py
```
