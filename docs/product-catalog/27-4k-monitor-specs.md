| Color Modes | sRGB, Paper, Low Blue Light, Custom |

### Notes on Color Accuracy

99% sRGB coverage is ideal for web design, UI/UX work, software development, and general professional use. For print production work requiring P3 or AdobeRGB coverage, a wide-gamut display should be considered. The factory calibration with Delta E < 2 means colors are accurate enough for professional web content without requiring additional calibration hardware.

## Connectivity

| Port | Specification | Notes |
|---|---|---|
| USB-C | 1x USB-C (65W Power Delivery) | One cable for video + laptop charging |
| HDMI | 2x HDMI 2.0 | 4K @ 60Hz |
| DisplayPort | 1x DP 1.4 | 4K @ 60Hz, supports DSC compression |
| USB-A | 2x USB-A 3.0 (downstream hub) | Powered via USB-C upstream connection |
| Audio Out | 1x 3.5mm headphone jack | — |

### USB-C Power Delivery Details

The USB-C port delivers 65W of power delivery, which covers:
- Laptop Pro 15" at full charge speed
- Most 13" and 15" laptops at full charge speed
- Tablets and phones at maximum charge speed

Connecting a compatible laptop via USB-C provides simultaneous: 4K video signal at 60Hz, USB hub functionality, and 65W charging — through a single cable.

### Multiple Source Switching

The monitor supports 2x HDMI inputs allowing you to connect a desktop and a laptop simultaneously, switching between them via the OSD menu or a dedicated input button.

## Setup Recommendations

- **Optimal viewing distance:** 24-35 inches from screen (60-90cm)
- **Color mode:** Set to sRGB for accurate web color reproduction
- **Brightness:** 150-200 nits is comfortable for office environments; reduce to 80-100 nits in dim rooms
- **Blue light filter:** Enable in evening hours to reduce sleep disruption

## Pairing Recommendations

- **Laptop Pro 15"** — Single USB-C cable for video + charging + USB hub
- **USB-C Hub 7-in-1** — Connect downstream from the monitor's USB-A ports for additional peripherals
- **Mechanical Keyboard** — Connect via monitor's USB-A hub for cleaner cable management
