| Carpet Boost | Automatic suction increase when carpet detected |

### LiDAR Navigation Advantage

LiDAR (Light Detection and Ranging) emits laser pulses and measures time-of-flight to build a precise 360° map of the environment. Benefits over older camera or infrared systems:

- **Works in the dark:** LiDAR is active illumination, not passive camera; no light required
- **Precise room mapping:** Walls and furniture are mapped to within a few centimeters
- **Efficient path planning:** Systematic back-and-forth cleaning pattern minimizes missed areas and redundant passes
- **Virtual boundaries:** App-drawn no-go zones and virtual walls are respected accurately

### First Run Setup

For optimal mapping accuracy:
1. Clear major obstacles from the floor (chairs pushed in, cables managed)
2. Allow the first mapping run to complete without interruption
3. Assign room labels in the app after the initial map is created
4. Create virtual no-go zones for areas with loose cables or fragile items

| Base Noise (emptying) | 70dB (approximately 15 seconds) |
| HEPA Bag Filtration | Yes — seals allergens inside bag during emptying |

### Self-Emptying System Notes

The base automatically empties the robot's dustbin after each cleaning run. Suction pulls debris from the robot bin into the 2.5L bag in the base. This eliminates daily manual emptying and contains allergens in a sealed bag until disposal.

The emptying process takes approximately 15 seconds and produces approximately 70dB of noise — louder than the robot itself but brief. Schedule cleaning at appropriate times if noise is a concern.

## Filtration

| Specification | Value |
|---|---|
| Robot Filter Type | HEPA-grade filter |
| Filtration Efficiency | 99.97% of particles ≥ 0.3 microns |
| Filter Replacement Interval | Every 2-3 months (wash monthly) |
| Self-Emptying Bag | HEPA filtration (allergens sealed during emptying) |

### HEPA Filtration Notes

The HEPA-grade filter in the robot captures fine dust, pollen, and pet dander. The filter is washable — rinse under water and allow to air dry completely (24 hours) before reinstalling. Never reinstall a damp filter; moisture can damage the motor.

For allergy and asthma households, replace the filter every 2 months regardless of wash frequency.
